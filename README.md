# 🛍️ RetailPulse: Retail Sales Analytics Dashboard

An end-to-end **Power BI Business Intelligence project** developed using the Online Retail Dataset. This project transforms raw retail transaction data into interactive dashboards that provide valuable business insights into sales, customers, products, profitability, inventory, and regional performance.

---

## 📌 Project Overview

RetailPulse is an interactive Power BI dashboard designed to help business stakeholders monitor sales performance and make data-driven decisions.

The project covers the complete BI workflow:

- Data Cleaning using Power Query
- Data Modeling
- DAX Measure Creation
- Interactive Dashboard Design
- Business Insight Generation

---

**Dataset:** Online Retail Dataset

The dataset contains:

- Invoice Transactions
- Product Details
- Customer Information
- Country
- Invoice Date
- Quantity
- Unit Price

# 📈 Business Insights

- United Kingdom generated the highest revenue.
- November recorded the highest monthly sales.
- Q4 contributed the strongest revenue performance.
- A small number of products generated the majority of revenue.
- High-value customers significantly contributed to overall sales.
- Revenue increased with higher quantities sold.
- Slow-moving products were identified for inventory review.

# 📌 Business Recommendations
